(static) deleteOfflineData(mac)

delete the offline data in devices.
  • This is an asynchronous call, it will return immediately.
  • If delete offline data successfully, following event will be emit:
    BG5LModule.Event_Notify("event_notify_bg5l")
    The key and value will be as below:
    KeyValue
    actionBGProfileModule.ACTION_DELETE_OFFLINEDATA
    BGProfileModule.DELETE_OFFLINEDATAtrue/false
  • If error happens, following event will be emit:
    BG5LModule.Event_Notify("event_notify_bg5l")
    The key and value will be as below:
    KeyValue
    actionAMProfileModule.ACTION_ERROR_BG("action_measure_error")
    BGProfileModule.ERROR_NUM_BG("error_num")BGProfileModule.ERROR_NUM_BG
    BGProfileModule.ERROR_DESCRIPTION_BG("error_description")Detailed description of the error
  • Attention, if you want to be notified, it is mandatory to call following method before you call this method:
    DeviceEventEmitter.addListener(BG5LModule.Event_Notify, function (e: Event){});
